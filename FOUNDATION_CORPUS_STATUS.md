This file is the current control table for RFOP foundation and foundation-method papers. It separates foundation papers from external exemplars, branch applications, version-lineage files, and internal construction documents.


No newer foundation paper is currently assumed beyond Declared Witness Assembly. Later files should be treated as applications, branch papers, versions, or internal documents unless explicitly reclassified after audit.

# Canonical Working Manuscripts

This folder contains public working manuscripts for papers that are canonical to the Relation-First Organization Programme, but not yet final journal-ready or standalone arXiv-ready article versions.

The distinction is intentional:

- **canonical** means the paper is part of the programme's core architecture or methodology;
- **working** means the manuscript still needs structural reorganization, reference stabilization, notation checks, or paper-register refinement before standalone submission.

These manuscripts are included here because branch and application papers explicitly depend on their methodology. They therefore need stable public repository paths.

## Citation use

Branch/application papers may cite these as repository working manuscripts when they depend on BVD, GBS, or DWA methodology.

Recommended citation status:

```text
Motruk, Artur. [Title]. Relation-First Organization Programme repository, canonical working manuscript, [commit/release tag when available].
```

## Non-final status

These are not final journal or arXiv versions. In particular:

- BVD and GBS still need structural reorganization to reduce incremental construction order.
- DWA needs further formula/source QA and reference stabilization.
- All three need full bibliography and branch-dependency checks before standalone preprint use.

They are public to establish programme dependency structure, not to claim final article polish.

## Folder policy

Use `papers/canonical_working/` for programme-canonical manuscripts that are public and citable by repository path, but still structurally working.

Use `papers/canonical/` only for public repository manuscripts that have passed the stricter public-register and cross-reference checks.

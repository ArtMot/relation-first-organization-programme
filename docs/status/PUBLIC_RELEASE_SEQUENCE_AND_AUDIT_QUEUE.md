# Public Release Sequence and Audit Queue

1. **Intellectual/dependency order** — how the programme is built.
2. **Public reader order** — how an outside reader should enter the corpus.
3. **Suitability cleanup order** — which files should be audited first for public release.
4. **Publishability order** — which externally legible papers may be easiest to submit to journals or arXiv.

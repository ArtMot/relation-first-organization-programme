## Release notes

This release establishes the public manuscript footprint for the Relation-First Organization Programme.

It includes:

- fourteen canonical repository manuscripts under `papers/canonical/`;
- three public canonical working manuscripts under `papers/canonical_working/`;
- a public manuscript index;
- citation metadata;
- release manifest;
- non-claims and scope statement;
- AI-assistance disclosure;
- public suitability and cross-reference protocols;
- status/provenance documents for staged development.

The public canonical working manuscripts are included because later branch/application papers depend on their methodology. They are citable by repository path, but they are not final standalone journal-ready or arXiv-ready article versions.

This release does not claim to derive General Relativity, the Standard Model, QED, physical constants, quantum mechanics, empirical dark-sector corrections, or a completed theory of nature. It also does not claim that generated candidates are verified solutions, that diagnostics prove external-domain theorems, or that local charted success implies global objecthood.

Suggested citation metadata is provided in `CITATION.cff`. A Zenodo DOI should be added after the release archive is deposited.

# Documentation Index

This folder holds release controls, status tables, public-audit protocols, provenance notes, and frontier/branch planning documents for the Relation-First Organization Programme.

The repository root is reserved for the public landing page, citation metadata, and the main manuscript index.

## Folders

| Folder | Purpose |
|---|---|
| `docs/release/` | Author metadata, non-claims/scope, AI disclosure, release checklist, release policy. |
| `docs/status/` | Corpus status tables, public title register, release sequence, citation matrix. |
| `docs/protocols/` | Suitability audit, cross-reference audit, manuscript export queue. |
| `docs/provenance/` | Version-lineage and internal archive notes. |
| `docs/frontier/` | Frontier-branch triage notes and branch-specific release cautions. |

## Public manuscripts

Use:

```text
PUBLIC_MANUSCRIPT_INDEX.md
```

for the stable public manuscript paths.

Use:

```text
papers/canonical/
papers/canonical_working/
```

for the actual public manuscript files.

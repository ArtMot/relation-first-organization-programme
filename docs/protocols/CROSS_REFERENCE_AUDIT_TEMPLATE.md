## Load-bearing dependency check

For each internal RFOP reference marked load-bearing:

1. Confirm whether the cited paper is included in the same public release.
2. If not, confirm whether the needed theorem/definition is summarized locally.
3. If not, decide whether the dependency can be downgraded to orientation rather than proof support.
4. If not, mark the manuscript not release-ready.

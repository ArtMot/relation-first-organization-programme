# Public Manuscript Index

This index records RFOP manuscripts currently available through stable public repository paths.

The index distinguishes between:

- `papers/canonical/`: repository manuscripts that have passed the stricter public-register cleanup used for the first public programme corpus;
- `papers/canonical_working/`: programme-canonical working manuscripts that are public because other RFOP branch/application papers depend on them, but which still require structural reorganization or further source/reference QA before standalone preprint or journal submission.

## Citation guidance

For repository manuscripts:

```text
Motruk, Artur. [Title]. Relation-First Organization Programme repository, repository manuscript, [commit/release tag when available].
```

For canonical working manuscripts:

```text
Motruk, Artur. [Title]. Relation-First Organization Programme repository, public canonical working manuscript, [commit/release tag when available].
```

Do not cite a pending family as a public manuscript until a stable path exists in this index.
